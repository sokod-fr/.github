# Sokod

**Software engineering studio for hard problems.**

We lead the engineering where it's hard: architecture, performance, applied
AI, and large-scale refactors and migrations. Senior-only, hands-on, embedded
in your team.

Run by a founder who ships — building software since 2007, now Founder & CTO
of City Map 3D. A hands-on operator, not an advisor on the sidelines.

### What we do

* **Architecture** — distributed systems, data modelling, API design.
* **Performance** — profiling, query and render optimization, scale.
* **Applied AI** — recommendation engines, semantic and vector search,
  embeddings, local-LLM and generative pipelines.
* **Front platforms** — Vue 3 at senior level, design systems, tooling.
* **Refactors & migrations** — untangling legacy without stopping the line.

### What we build

* 🗺️ **[City Map 3D](https://citymap3d.com)** — a proprietary real-time 3D map
  platform on OpenStreetMap. 1B+ rows, custom WebGL engine, operated in production.
* 🧪 **Sokod Labs** — tools and experiments, built in-house.
